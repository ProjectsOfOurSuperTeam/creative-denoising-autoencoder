# Frontend

Фронтенд для презентації проєкту на базі Streamlit.

## Розділи

- `Завантаження` — форма завантаження, вибір моделі, перегляд результату
- `Порівняння` — порівняння одного зображення між кількома моделями
- `Про проєкт` — короткий опис цілей та метрик

## Запуск

1. Встановити залежності:
   - `pip install -r requirements.txt`
2. Запустити застосунок:
   - `streamlit run frontend/app_streamlit.py`
3. Відкрити URL, який покаже Streamlit у терміналі
